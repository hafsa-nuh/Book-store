## PROJECT DESCRIPTION
This ia a website that is based on Book store/shop. The search asks the user to input the author name or book title to select the book. The User logo is for form submition for the website.

## SETUP
to access  this project on your local files, you can clone it using this steps

```bash
  Open the Terminal

Clone the repository: git@github.com:hafsa-nuh/Book-store.git

Change Directory: cd Book-store

Open index.html file on your browser
```
